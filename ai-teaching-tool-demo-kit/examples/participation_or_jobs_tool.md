# Participation Or Class Jobs Tool

## Teaching Goal

Help organize participation, rotating class responsibilities, peer roles, or classroom tasks in a transparent way.

## Classroom Flow

The instructor defines jobs or participation opportunities, students select or are assigned roles, and the tool tracks completion.

## Student Actions

Students may sign up for jobs, rank preferences, report completion, or submit participation actions.

## Instructor Actions

Create jobs, set capacity, approve assignments, record completion, adjust records, and export data.

## Data Saved

Timestamp, session ID, participant ID, job or action ID, status, points or credit if used, and instructor notes if needed.

## Possible Scoring Or Payoff Logic

Use completion counts, simple points, rotating priorities, or market-like credits. Keep rules visible.

## What To Test

Test duplicate signups, capacity limits, edits, exports, private notes, and reset behavior.

## Possible Extensions

Add random assignment, preference matching, waitlists, audit logs, or dashboard summaries.
