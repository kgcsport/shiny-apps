# Public Goods Game

## Teaching Goal

Help students see the tension between private incentives and group outcomes.

## Classroom Flow

The instructor creates groups, sets an endowment and multiplier, starts a round, collects contributions, then reveals group totals and payoffs.

## Student Actions

Students choose how many tokens to contribute to a group account.

## Instructor Actions

Set parameters, start rounds, advance rounds, reveal results, reset test data, and export records.

## Data Saved

Raw contribution events with timestamp, session ID, round number, participant ID, group ID, contribution, endowment, multiplier, and payoff.

## Possible Scoring Or Payoff Logic

Payoff equals tokens kept plus an equal share of multiplied group contributions.

## What To Test

Test missing choices, payoff calculations, group assignment, public display privacy, reset behavior, and raw data export.

## Possible Extensions

Add multiple rounds, random rematching, communication treatments, anonymous group histories, or treatment comparisons.
